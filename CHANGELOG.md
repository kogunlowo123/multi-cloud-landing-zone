# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-07

### Added
- Initial multi-cloud landing zone modules for AWS, Azure, and GCP
- Identity federation module for cross-cloud SSO and RBAC
- AWS landing zone with Organization, SCPs, and Control Tower integration
- Azure landing zone with Management Groups, Policy, and Blueprints
- GCP landing zone with Organization, Folders, and Org Policies
- Comprehensive README with Mermaid architecture diagram
